# Задание 3. Выпуклое программирование

Номер задания выбирается по формулам  
Целевая функция:  
$(N_1(N_2+14) mod 40)+1$  
ограничения:  
$(N_1(N_2+15) mod 60)+1$  
Здесь $N_1$ - номер группы, $N_2$ - номер студента в списке группы
